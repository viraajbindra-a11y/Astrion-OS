# 🧪 Tester — Starter Task

## What to send them

> Hey! Thanks for picking Tester. Here's your first 15-minute task.
>
> **Goal:** Install Astrion OS on your laptop (or in a VM) and tell me what the first 5 things that break, feel broken, or feel confusing are. Be brutally honest — the meaner you are the better Astrion gets.
>
> **Step 1 — Download it:**
> Go to https://github.com/viraajbindra-a11y/Astrion-OS/releases/latest
> Download `astrion-os-0.1.99-amd64.iso` (it's about 1.8 GB — might take a few minutes on slow WiFi).
>
> **Step 2 — Install or boot it:**
> Easiest way: install **UTM** (https://mac.getutm.app if you're on Mac, otherwise VirtualBox is free for Windows) and make a new Linux VM that boots from the ISO. You don't need to actually install it to your computer — just boot it in the VM.
>
> **Step 3 — Use it for 10 minutes:**
> Click around. Open apps. Try the Search (Ctrl+Space). Open Notes and write something. Try a few games. Try to break things on purpose.
>
> **Step 4 — Send me a list of 5 things:**
> Could be anything:
> - "X crashed when I clicked Y"
> - "The calculator has a weird button"
> - "I couldn't figure out how to Z"
> - "The font looks tiny"
> - "Terminal froze when I did X"
>
> Screenshots are GOLD — if you can screenshot a bug, it's 10x more helpful. On Mac press Shift+Cmd+4, drag over the area, then send me the image.
>
> Thank you seriously. This is the single most useful thing anyone can do for Astrion right now.

## What Viraaj does with the result

- Open `tasks/lessons.md` and add any new bugs found as lessons
- For each bug, decide: fix now (if trivial) or add to a backlog file
- If the friend found something really weird, ask follow-up questions
- **CREDIT THEM in the next release notes** — "Bug reports from Koa/Naren/Lauren"

## Follow-up task after this one lands

Ask them to install the NEXT version too and see if the bugs they reported are fixed. That closes the loop and makes them feel like part of the process.
