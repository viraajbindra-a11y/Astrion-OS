## What I learned (lessons added: #166-#168)

- **#166** — Audit your audit. The "20+ apps unguarded localStorage" complaint was already fixed at the substrate level (`safe-storage.js` monkey-patches Storage.prototype globally). I re-flagged work that was already done. Always check the existing fix before scoping new work.
- **#167** — "AI-native OS" is no longer a blue-sky claim by 2026-Q2. Brain Tech, OpenClaw, MS Copilot+, Apple Intelligence, Anthropic Cowork, VAST Data, AIOS — every player calls themselves an AI OS. The differentiation is the safety triple, not the category name.
- **#168** — Edit-tool emoji escape mismatch. The Edit tool tries both rendered emoji and `\uXXXX` escape forms but can fail on long blocks where the source uses literal escapes. Use Python sed-style replace for surgical patches when blocks contain mixed emoji/escape content.
