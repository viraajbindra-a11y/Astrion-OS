## How to add a new wallpaper

1. Drop the file in this directory (PNG or JPG for photos, SVG for generated art).
2. Add an entry to the `wallpapers` array in `js/apps/settings.js`.
3. Add a row to this CREDITS.md file with the source and license.
4. If contributed, update `tasks/contributions.md` with the contributor and
   revenue-share note.
5. Keep files under 2 MB where possible — they get bundled into the ISO and
   the GitHub release has a 2 GB total cap per asset.
