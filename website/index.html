The Safety Triple

Three rules. All mandatory. Skip any one and the rest break.

This is the only thing in the AI-OS market that no competitor ships. Microsoft, Apple, OpenAI, Anthropic — they have pieces. Astrion has the stack.

Verifiable

Receipts on every artifact

Ask Astrion to build an app — it writes the spec first, then tests against the spec, then code that passes the tests. Every artifact ships with a provenance chain: which model wrote it, which seed, which tests it passed. Code app.bundle refuses unless tests pass and the spec is frozen.

Reversible

Branch + rewind any L2 action

Every dangerous thing happens in a practice universe first. You see the diff, then say "yes for real." Type branches in Spotlight to see every committed branch with a ⏪ Rewind button. Mistakes are never permanent — except for actions explicitly marked point-of-no-return, which require typing the capability id to confirm.

Socratic

It asks before doing big stuff

Not "are you sure?" popups. Real questions: "you said you wanted X, did you think about Y?" A second AI red-teams every L2+ plan in a different model and shows risks side-by-side with the planner's intent. A rubber-stamp tracker watches for "approved in <1.5s" and warns you when you stop reading. The AI never trains you to stop deciding.

Under the hood

Architecture, not bolt-on.

The kernel knows about intents and capabilities. The shell knows about agents and gates. Every layer was built knowing AI was here.

🧠

Intent Kernel

You say what you want, not what to do. The kernel decomposes intents into capability calls, executes them with budget tracking, halts on failure with an explicit step list. Spotlight, the chat panel, and skills all flow through it.

🤖

Dual-Process Brain

System 1 is local Ollama, always on, handles reflexes. System 2 is Claude or OpenAI, called on uncertainty. A calibration tracker watches per-category accuracy and escalates topics permanently when S1 falls below 70%.

🪟

Native C/GTK3 Shell

Each app opens in its own real GTK window. Menubar, dock, desktop are native widgets. Battery, Wi-Fi, and volume read from the actual hardware. 4068 lines of C — not a browser pretending to be an OS.

📐

Hypergraph Storage

Notes, todos, reminders, contacts — all nodes in one graph. Spotlight queries it. Apps read from it. Every mutation is a commit, undoable for 30 days. Goodbye files-and-folders, hello structured everything.

🧩

Skill DSL (M7)

Five keywords: goal, trigger, when, do, constraints. Phrase, cron, or event triggers. Per-skill capability level + token budget. Bundled 20, install more by paste.

🔌

Capability Tiers

L0 observe → L1 sandbox → L2 real → L3 self-mod. Every action declares its level, reversibility, and blast radius. The AI never promotes itself across tiers. You stay in control.
