Features

Not just a demo. A real operating system.

Every feature you'd expect from a desktop OS, plus AI capabilities that no other OS has.

🧠

Smart Spotlight Search

Type anything. Convert units, check time zones, preview colors, flip coins, generate UUIDs, calculate math — all instant, all offline, no AI key needed.

🤖

Dual-Process AI Brain

System 1 (fast, local) handles easy stuff. System 2 (careful, cloud) handles hard stuff. The boundary shifts as AI gets smarter. Your OS evolves with the tech.

🪟

Real Window Management

Drag to edges for snap layouts. Pin on top. PiP mini-mode. Grid tile. Cascade. Corner snap for quadrants. Alt+Scroll for transparency. All with keyboard shortcuts.

🌐

Built-in Web Browser

Browse real websites with a server-side proxy that makes iframe embedding actually work. Tab bar, bookmarks, back/forward, YouTube embed.

📝

Full-Featured Apps

Text editor with syntax highlighting, find bar, and VS Code shortcuts. Notes with markdown preview. Finder with breadcrumbs and sort. Calculator with history.

🔒

Safety by Design

Capability tiering (L0-L3), irreversibility budgets, diff previews before writes, Socratic prompts. The AI never promotes itself — you stay in control.

🎮

Games with AI Autoplay

Snake, Chess, 2048, Tetris, Minesweeper — each with an AI autoplay button that uses pathfinding and evaluation algorithms. Watch the AI play.

🎨

Customizable Everything

8 accent colors, wallpaper picker, window opacity, dock toggle, virtual desktops (Spaces), focus/DND mode, keyboard shortcuts overlay.

📋

Smart Clipboard

Clipboard history (Ctrl+Shift+V) auto-detects URLs, emails, colors, JSON, code, phone numbers. Color entries show a live swatch preview.
